# xless

xless mainly include three class and h5 reset file
## We have learned much from/been inspired by/taken code where offered from:
    a. HTML5-Reset
    b. Animate.css

## Include the xLess on your less file top
```html
    @import "xLess"
```

ok , Let us start using it
